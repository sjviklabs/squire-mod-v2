**CRITICAL validator rule:** Every `defineInRange(name, default, min, max)` call MUST have `min <= default <= max`. A default outside its own range causes an infinite correction loop on every server start. Example of the bug: `defineInRange("followStart", 6.0, 8.0, 32.0)` — default 6.0 fails min 8.0. Every entry below must be verified.

**Conservative ATM10 defaults:** Lower than v0.5.0 where applicable. ATM10 is a heavily modded pack — lower tick rates and shorter ranges reduce performance impact.
