private void applyModifiers() {
    SquireTier currentTier = SquireTier.fromLevel(currentLevel);

}

3. XP THRESHOLD FROM LOADER in recalculateLevel():
```java
private void recalculateLevel() {
    int xpRemaining = totalXP;
    int newLevel = 0;
    // Walk tier thresholds from loader; fall back to SquireTier.minLevel
    for (SquireTier tier : SquireTier.values()) {
        Optional<TierDefinition> def = ProgressionDataLoader.getTierDefinition(tier);
        int xpToNext = def.map(TierDefinition::xpToNext).orElse(0);
        if (xpToNext > 0 && xpRemaining >= xpToNext) {
            xpRemaining -= xpToNext;
            newLevel = tier.getMinLevel() + 1; // advance past this tier threshold
        }
    }
    // Clamp to Champion max level (30)
    newLevel = Math.min(newLevel, SquireTier.CHAMPION.getMinLevel());
    if (newLevel != currentLevel) {
        currentLevel = newLevel;
        squire.setSquireLevel(currentLevel);
        onLevelUp();
    }
}
```
