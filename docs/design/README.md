# Squire — Design Package

Design assets for the Squire Mod's distribution push (Modrinth, CurseForge, future landing page).

## Files

- `index.html` — interactive design file with three tabs (Identity, Logo, Listing) and a Tweaks panel. Open in any browser. No build step.
- `handoff/` — plain-markdown deliverables for humans:
  - `README.md` — start here, 5-minute overview
  - `BRAND.md` — colors, type, motif rules (single source of truth)
  - `LISTING_COPY.md` — paste-ready copy for Modrinth / CurseForge descriptions
  - `SCREENSHOTS.md` — in-game capture guide
  - `LOGO_EXPORTS.md` — export checklist for the Seal mark
  - `NEXT_ROUND.md` — prompts for round 2 (landing page + in-game UI)
