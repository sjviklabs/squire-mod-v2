# Squire — Visual Identity

**One-liner:** Medieval-heraldic with a tactical-dry undertone. A hired henchman, not a pet.

## Palette

| Token | Hex | Role |
|---|---|---|
| Ink | `#0F0E0B` | Primary ground (dark UI, hero backgrounds) |
| Slate | `#252119` | Panels, cards, secondary ground |
| Vellum | `#E8DFC9` | Body text on dark; light ground when inverted |
| Brass | `#B89558` | **Primary accent** — 90% of accent load |
| Claret | `#7A2A2A` | Alert / error / destructive |
| Moss | `#4B5A3E` | Live / success / growth state |

**Rule:** Brass carries nearly everything. Claret and Moss are reserved for state — don't decorate with them.

## Typography

| Role | Family | Weight | Notes |
|---|---|---|---|
| Display | **Cinzel** | 600–700 | Headlines, wordmark, buttons. Wide tracking (0.08–0.3em). |
| Editorial | **Cormorant Garamond** | 500 italic | Pull-quotes, lede, hero taglines. |
| UI / Body | **Inter** | 400 / 600 | Paragraphs, UI labels, metadata. |
| Mono | **JetBrains Mono** | 500 | Commands, spec labels, eyebrows. |

All four are on Google Fonts (free, permissive). No custom licensing needed.

## Motifs

Three reusable ornaments (see Identity tab in the design file):

1. **Heraldic Seal** — circular seal with inner shield. Used for the primary mark.
2. **Trades Mark** — crossed pickaxe + sword. Used where combat/work duality needs emphasis.
3. **Section Rule** — thin horizontal rule with small central diamond. Use between body sections in long-form docs.

**Draw rules:**
- Stroke width 1.0–1.4 at display size.
- No fills except for solid shields/hearts of the mark.
- Always monochromatic — brass on dark, ink on vellum. Never gradient.

## Voice

- Terse. Sentences end.
- No exclamation points.
- In-character occasionally — Squire speaks of itself in third person in docs: *"The squire deposits."* not *"It will deposit."*
- Never cute, never ironic. Dry humor is allowed; comedy is not.

## What to avoid

- Rounded corners above 2px.
- Gradients (except subtle radial wash on hero).
- Drop shadows on UI elements.
- Gold foil, sparkle effects, or anything that reads as "fantasy product."
- Emoji.
- Green accents in general — that territory belongs to SJVIK Labs; keep Squire distinct.
