# Squire — Round 2 Briefing

Once v3.0.1 is live on Modrinth and CurseForge, the next two design pieces are the **landing page** and the **in-game UI mocks**. Here's how to brief them.

## Landing page (squire-mod.com or similar)

**Why:** SEO. Modrinth and CurseForge don't rank as well as a dedicated page for queries like "Minecraft NeoForge companion mod." A landing page captures that traffic and sends it to the distribution listings.

**Prompt to paste:**

> Using the brand system in `Squire Design.html` and the copy in `handoff/LISTING_COPY.md`, design a single-page landing site for Squire. Include:
> - Hero (wordmark, tagline, install CTA to Modrinth + CurseForge, looping video if available)
> - Feature grid (pull directly from the listing)
> - A "how it works" section — 3 steps, each with a screenshot: hire, equip, assign
> - A commands reference block
> - Roadmap (3 upcoming features, pulled from my issues list)
> - Footer with source, license, contact
>
> Full-bleed prototype with desktop + mobile at 375px. Same heraldic system. No new colors.

## In-game UI mocks

**Why:** The mod is functional but the chat-command interface is the only way to drive it. A GUI would make it approachable for the 90% of players who won't learn commands.

**Prompt to paste:**

> Mock up an in-game **Squire's Ledger** GUI — the screen that opens when you right-click the squire. Show 3 variations:
> 1. **Minecraft-native** — standard container-slot layout with the squire's inventory, armor slots, role selector, and a compact status readout. Matches vanilla MC UI conventions.
> 2. **Patchouli-style page** — looks like a page of the in-game book, but interactive. More narrative, less grid.
> 3. **Heraldic panel** — uses the brand system; parchment background, seal in the corner, brass rules. Most "branded" option.
>
> For each, show the ledger both empty and populated with gear + level 12 stats. Same 1920×1080 design frame, minecraft-style 2x pixel density where needed.

## Things worth considering for round 2

- **Trailer video.** A 30–45s cut of gameplay set to the Cinzel wordmark reveal. Converts better than anything else on storefronts. I can storyboard it.
- **Patron/support page.** If you're taking sponsorship, a simple "Become a patron" page using the same system. Low effort, high signal.
- **Press kit.** A single zip with logo exports, screenshots, and short/long descriptions. If any modpack YouTuber picks it up you want this ready in 30 seconds.
- **Twitter/Bluesky launch assets.** 1200×675 cards in the brand system — one per feature.

## What to re-check before round 2

- Any changes from v3.0.1 → next version (new commands? role changes?)
- Real download numbers once listings go live — the hero's "Downloads" stat currently reads `— —`.
- Whether "Paid in coin or in kind" stays as the tagline or we workshop it more.
