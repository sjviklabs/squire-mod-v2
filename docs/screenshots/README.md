# Squire — Screenshot Gallery

In-game captures used by the Modrinth/CurseForge listings and the landing page.

See `../design/handoff/SCREENSHOTS.md` for the capture plan — shot list, settings, file-naming conventions.

When adding shots here, keep them as:
- PNG, no JPEG (Minecraft's UI doesn't JPEG well)
- 1920×1080 minimum (Modrinth auto-scales; CurseForge wants large source)
- F1 to hide HUD for "hero" shots; keep HUD for "gameplay proof" shots
